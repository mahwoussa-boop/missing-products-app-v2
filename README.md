# 🌬️ نظام المنتجات المفقودة الذكي — متجر مهووس

> اكتشف الفرص الذهبية — تحقق بدقة — أرسل بأمان إلى Make.com

## نظرة عامة

تطبيق Streamlit مستقل لاكتشاف المنتجات المتوفرة عند المنافسين وغير المدرجة في متجر مهووس، مع محرك تحقق ذكي متعدد الطبقات لمنع التكرار.

## البنية

```
missing_products_app/
├── app.py              # الواجهة الرئيسية (Streamlit)
├── ai_matcher.py       # محرك المطابقة الذكي (5 طبقات)
├── db_manager.py       # تحميل وتنظيف البيانات
├── make_sender.py      # إرسال آمن إلى Make.com
├── config.py           # الإعدادات والمرادفات والماركات
├── requirements.txt    # المكتبات المطلوبة
├── aliases.json        # ذاكرة الأسماء البديلة (يُنشأ تلقائياً)
└── .streamlit/
    ├── config.toml     # إعدادات المظهر
    └── secrets.toml    # المفاتيح السرية (لا يُرفع لـ Git)
```

## محرك المطابقة — 5 طبقات

1. **تطبيع عدواني** — توحيد الأسماء عربي/إنجليزي (100+ مرادف للماركات والأنواع)
2. **استخلاص المكونات** — الماركة، الحجم (ml)، النوع (EDP/EDT)، تستر/أصلي، طقم
3. **مطابقة ذكية** — `rapidfuzz.token_set_ratio` مع مكافأة تطابق الماركة وعقوبة اختلاف الحجم
4. **كشف البدائل** — هل لدينا التستر فقط؟ حجم مختلف؟ نوع مختلف؟
5. **ذاكرة الأسماء (Alias Memory)** — تتعلم من مراجعاتك السابقة

## مستويات الثقة

| المستوى | الوصف | الإجراء |
|---------|-------|---------|
| 🟢 مفقود مؤكد | نسبة تشابه < 55% مع كل منتجاتنا | جاهز للإرسال إلى Make.com |
| 🟡 يحتاج مراجعة | تشابه 55-75% أو وجود مشابه | راجع يدوياً (زر ✅ أو ❌) |
| 🔴 مشبوه | تشابه عالٍ مع منتج مختلف النوع | لا يُرسل — مراجعة فقط |
| ✅ موجود | تشابه ≥ 75% بعد التطبيع العدواني | تم التحقق — لا حاجة لإضافته |

## الملفات المطلوبة

### ملف متجر مهووس (CSV)
- **الصف الأول**: تسمية عامة (يُتجاهل)
- **الصف الثاني**: الـ Header الحقيقي
- الأعمدة: `أسم المنتج` | `سعر المنتج` | `صورة المنتج` | `رمز المنتج sku` | `الماركة`

### ملفات المنافسين (CSV)
- يتم كشف الأعمدة تلقائياً (اسم المنتج، السعر، الصورة، الرابط)
- اسم المنافس يُستخلص من اسم الملف
