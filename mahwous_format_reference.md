# تنسيق وصف مهووس المرجعي (من صفحة 6 روز)

## هيكل الوصف الكامل:

### 1. H2 - اسم المنتج بالعربي والإنجليزي
```
عطر 6 روز أو دو بارفيوم 100مل (نسائي)
6 Rose Eau de Parfum 100ml (For Women)
```

### 2. مقدمة عاطفية (فقرة افتتاحية)
- جملة تسويقية عاطفية مثيرة للاهتمام
- لا إيموجي
- 3-4 جمل

### 3. H3 - تفاصيل المنتج (قائمة نقطية)
```
- الماركة: [اسم الماركة بالعربي والإنجليزي] مع رابط داخلي
- النوع: عطر
- الجنس: نسائي/رجالي (For Women/Men)
- الحجم: [X] مل
- التركيز: أو دو بارفيوم (Eau de Parfum)
- سنة الإصدار: [السنة أو غير متوفر]
- الخط العطري: [الخط]
```

### 4. H3 - رحلة العطر: النفحات والمكونات
```
- مقدمة العطر: [مكونات القمة]
- قلب العطر: [مكونات القلب]
- قاعدة العطر: [مكونات القاعدة]
```

### 5. H3 - لماذا تختار/تختارين [اسم العطر]؟
- 4 نقاط مميزات بعنوان **بولد** وشرح

### 6. H3 - الأسئلة الشائعة (FAQ)
- سؤالان على الأقل بصيغة "س:" و"ج:"

### 7. H3 - اكتشف أكثر من مهووس للعطور
- روابط داخلية مخفية تحت الكلمات (Hyperlinks)
- مثال: تصفحوا مجموعتنا من [العطور النسائية](رابط).

### 8. خاتمة تسويقية
- جملة ختامية تحفيزية
- فقرة "تجربة تسوق استثنائية بانتظرك في مهووس!"

## ملاحظات تنسيق مهووس:
- لا إيموجي في الوصف
- الروابط الداخلية مخفية تحت الكلمات (Hyperlinks) وليست ظاهرة
- الوصف بالكامل بالعربي
- الاسم الإنجليزي للمنتج يذكر مرة واحدة في البداية
- المكونات الحقيقية من Fragrantica
- HTML نظيف بدون CSS إضافي
